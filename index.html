LinkedIn Extension

Use the Chrome extension to save leads from LinkedIn profiles and search results directly to your pipeline.

  1. Install the Chrome extension from the LinkedIn Tools page.
  2. Open a LinkedIn profile or search results page, click the extension icon.
  3. The extension extracts the person's name, title, company, and LinkedIn URL.
  4. Configure your Supabase URL and API key in the extension popup, then click Save to ShowToShip.
Need the Supabase URL or anon key? Contact your admin.

Guides

How to use the ShowToShip system

📋 Getting Leads Into the System

Option A — Scraper (recommended for supported sites):

  1. Go to Tradeshows and click the Scrape button next to a show, or click + New Scrape Job on the All Leads page.
  2. Type the show name (it auto-creates a new show if one doesn't exist).
  3. Paste the exhibitor list URL(s) — one per line. Supported domains: mapyourshow.com, a2zinc.net (and custom A2Z domains).
  4. Toggle LLM Extraction on for unsupported sites (uses AI to parse the page). Keep it off for supported sites.
  5. Toggle Deep Scrape if you want to follow each exhibitor's detail page for more data.
  6. Toggle Find Emails to hunt for email addresses on company websites (slow on large lists).
  7. Click Run Scraper — results appear in the table automatically.

Option B — Manual CSV Import:

  1. Click Import CSV in the All Leads toolbar.
  2. Select a CSV file with columns like company name, contact, email, phone, website, etc.
  3. Map the detected columns to the right fields in the modal, then confirm.

Option C — LinkedIn Extension:

  1. Install the Chrome extension from the LinkedIn Tools page.
  2. Open a LinkedIn profile or search results page, click the extension icon.
  3. The extension extracts the person's name, title, company, and LinkedIn URL.
  4. Configure your Supabase URL and API key in the extension popup, then click Save to ShowToShip.
📊 Managing Leads
  • Filter by Show: Click a show name on the Dashboard (upcoming shows) or Tradeshows page to jump to all leads for that show.
  • Pipeline Stages: Use the left sidebar pipeline filters (New → Contacted → Qualified → Opportunity) to track progress.
  • Assign: Click a lead to open the detail panel, then pick a team member from the Assign dropdown.
  • Bulk Actions: Select multiple leads with checkboxes, then use Bulk Assign or Bulk Delete.
  • Letter Bar: Click a letter to filter leads alphabetically by company name.
  • Duplicate Finder: Click Find Duplicates to highlight leads with the same company name so you can merge or delete them.
  • Export: Click Export Excel in the sidebar to download all filtered leads as an .xlsx file.
🧹 Clearing Data
  • Clear leads per show: On the Tradeshows page, click Clear Leads next to a show to remove all leads from that show only.
  • Delete individual leads: Open a lead's detail panel and click Delete at the bottom.
  • Bulk delete: Select leads with checkboxes, click Bulk Delete, then confirm.

⚠ Deletions are permanent and cannot be undone.

🔧 Configuration

Go to Settings in the sidebar to configure:

  • GitHub Connection: Personal Access Token + repo name — used if you want to commit scrape results.
  • Supabase Database: Project URL and Anon Key — connects the app to your Supabase backend.
  • Admin Function URL: Edge Function for user management (admin only).
  • Team Members: Add names so you can assign leads to specific people.
🔧 Running Scrapers via GitHub Actions

For large or automated scrapes, use the GitHub Actions workflow:

  1. Go to the repo's Actions tab → Scrape Exhibitors workflow.
  2. Click Run workflow and fill in the show name and URL(s).
  3. The workflow runs the Playwright scraper in the cloud and saves results directly to Supabase.
  4. The SUPABASE_KEY secret must be set to the service_role key (not anon) for INSERT permission.

Supported sites: MapYourShow, A2Z Events (including custom domains), SMM (API), EuroNaval.
